# 🛒 Superstore Sales Analysis

## 📋 Overview
Analyzed 9,994 retail transactions using 
Google BigQuery SQL to identify revenue 
patterns and logistics optimization 
opportunities.

## 🛠️ Tools Used
- Google BigQuery (SQL)
- Tableau Public (Dashboard)
- GitHub (Documentation)

## ❓ Business Questions
1. Which category generates most revenue?
2. Which region performs best?
3. What are top 10 best selling products?
4. Which shipping mode is most popular?
5. Which customer segment drives most sales?
6. What is the monthly sales trend?

## 📊 Key Findings
- Technology drives highest revenue ($836K+)
- West region leads with $710K in sales
- Standard Class used by 60% of customers
- Consumer segment generates most revenue
- Q4 shows highest seasonal sales spike

## 💡 Business Recommendations
1. Focus inventory on Technology category
2. Expand West region operations
3. Optimize Standard Class shipping costs
4. Target Consumer segment in marketing
5. Prepare extra inventory for Q4 season
